Make your site super smooth, WP Smoother has features such as: smooth mouse wheel scrolling, smooth anchor scrolling and page load fading animations!

== Description ==

This simple yet elegant plugin will allow you to make your site super smooth!  
All features are editable and can be removed or changed to fit your needs.

Breakdown of features below:  
  
**Smooth Mouse Wheel Scrolling**

*   Enable / disable functionality
*   Editable scroll distance ( in pixels )
*   Editable scroll animation time ( in milliseconds )

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

All features will automatically enabled and set to the recommended value, navigate to the settings tab to update the options.
